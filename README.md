# Amy Ariyo Portfolio

Personal portfolio showcasing my work in Earth Sciences, GIS, remote sensing, environmental data analysis and scientific communication.

I recently completed an MSci in Earth Sciences at UCL, where my dissertation on Arctic sea-ice remote sensing received the department’s 2026 Research Project Prize.

## Featured work

- Arctic sea-ice radar analysis using satellite data
- Urban land-cover classification with Sentinel-2 imagery
- UK rainfall extremes analysis using ERA5 climate data
- CO₂ emissions visualisation using World Bank data
- Geological and environmental fieldwork across the UK and Germany

## Tools

Python, R, QGIS, remote sensing, statistical analysis and data visualisation.
